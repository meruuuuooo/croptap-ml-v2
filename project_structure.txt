=====================================
DIRECTORY SUMMARY
=====================================

Total Files: ~35+ (excluding __pycache__ and .git)
Total Size: ~90+ MB (mainly datasets and model files)

Key Directories:
- app/          : FastAPI application code (modular architecture)
- models/       : Trained ML models and datasets
- raw_datasets/ : Source CSV files for training and recommendations
- notebooks/    : Jupyter notebooks for ML development lifecycle
- deployment/   : Production deployment scripts and configs

Architecture:
- Hybrid recommendation system (40% rule-based + 60% ML)
- RESTful API using FastAPI
- Service-oriented modular design
- Clear separation of concerns (models, services, utils)

Main Entry Points:
- app/main.py                    : API server
- notebooks/01-05_*.ipynb        : ML training pipeline
- deployment/deploy.sh           : Production deployment

API Endpoints:
- POST /api/v1/recommend         : Get crop recommendations
- GET /health                    : Health check
- GET /docs                      : Interactive API documentation
